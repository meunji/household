# Supabase 프로젝트 확인 가이드

Supabase URL에 접속할 수 없는 경우 확인 방법입니다.

## 즉시 확인 사항

### 1. Supabase 대시보드 확인

1. [Supabase 대시보드](https://app.supabase.com) 접속
2. 로그인
3. 프로젝트 목록 확인
4. 프로젝트가 있는지 확인:
   - ✅ 프로젝트가 있음 → 프로젝트 URL 확인
   - ❌ 프로젝트가 없음 → 프로젝트가 삭제되었을 수 있음

### 2. 프로젝트 URL 확인

Supabase 대시보드에서:
1. 프로젝트 선택
2. **Settings** → **API** 메뉴
3. **Project URL** 확인:
   - 현재 URL: `https://fqgcxjddhddcrbazuseu.supabase.co`
   - 대시보드의 URL과 일치하는지 확인

### 3. 프로젝트 상태 확인

Supabase 대시보드에서:
1. 프로젝트 선택
2. 프로젝트 상태 확인:
   - ✅ Active (활성)
   - ⚠️ Paused (일시 중지)
   - ❌ Deleted (삭제됨)

## 가능한 원인

### 원인 1: 프로젝트가 일시 중지됨

Supabase 무료 플랜은 비활성 기간 후 프로젝트가 일시 중지될 수 있습니다.

**해결**:
1. Supabase 대시보드 접속
2. 프로젝트 선택
3. **Restore** 또는 **Resume** 클릭
4. 프로젝트 재활성화 대기

### 원인 2: 프로젝트가 삭제됨

프로젝트가 삭제되었을 수 있습니다.

**해결**:
1. 새 프로젝트 생성
2. 환경 변수 업데이트
3. 데이터베이스 스키마 재생성

### 원인 3: URL 변경

Supabase 프로젝트 URL이 변경되었을 수 있습니다.

**해결**:
1. Supabase 대시보드에서 현재 URL 확인
2. 환경 변수 업데이트
3. 재배포

## 다음 단계
